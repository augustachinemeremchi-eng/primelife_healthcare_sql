# PrimeLife Healthcare SQL Database

## Project Overview
A relational database built in MySQL to manage core healthcare operations at PrimeLife University Teaching Hospital.

## Tables
- **patients** – patient records and demographics
- **doctors** – doctor profiles and specializations  
- **diagnoses** – patient diagnosis records linked to doctors

## Skills Demonstrated
- Database design and table creation
- JOIN queries across multiple tables
- GROUP BY, HAVING, and ORDER BY clauses
- Healthcare data management

## Tools Used
- MySQL Workbench
- SQL
